# 李佳禧 · 个人网站 / Lexi Li · Personal Website

我的个人网站，包含教育经历、工作经历、个人爱好与成就，支持中英文一键切换和深浅色主题。纯静态页面（HTML / CSS / JS），无需任何构建工具。

My personal website featuring education, experience, hobbies, and achievements, with one-click Chinese/English switching and light/dark themes. Fully static (HTML / CSS / JS), no build step required.

## 本地预览 / Local preview

```bash
python3 -m http.server 8000
# 打开 / open http://127.0.0.1:8000
```

或直接用浏览器打开 `index.html`。 / Or just open `index.html` in a browser.

## 文件 / Files

- `index.html` — 页面结构与全部双语内容 / page structure and all bilingual content
- `styles.css` — 样式、配色、响应式、深色模式 / styles, colors, responsive, dark mode
- `script.js` — 中英切换、主题切换、滚动动效 / language & theme toggle, scroll animations
- `avatar.jpg` — 头像 / profile photo
